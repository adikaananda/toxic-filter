Install dalam 6 Langkah Mudah

Ikuti panduan step-by-step untuk menginstall extension di Chrome Anda

1

Download Extension

Klik tombol "Download Extension" di bagian atas halaman ini. File ZIP akan otomatis terdownload ke folder Downloads Anda.

üí° Tips: Pastikan koneksi internet Anda stabil untuk proses download yang lancar.
2

Extract File ZIP

Buka folder Downloads, klik kanan pada file ZIP yang telah didownload, lalu pilih "Extract Here" atau "Extract to Toxic Filter/".

Ingat lokasi folder hasil extract ini, karena Anda akan membutuhkannya di langkah selanjutnya.

3

Buka Chrome Extensions

Ada dua cara untuk membuka halaman Extensions di Chrome:

Cara 1: Ketik di address bar

chrome://extensions/

Cara 2: Melalui Menu Chrome

Klik icon titik tiga (‚ãÆ) di pojok kanan atas ‚Üí Extensions ‚Üí Manage Extensions

4

Aktifkan Developer Mode

Di halaman Chrome Extensions, cari toggle switch "Developer mode" di pojok kanan atas, lalu aktifkan (warna biru).

5

Load Unpacked Extension

Setelah Developer mode aktif, akan muncul tombol-tombol baru di bagian atas. Klik tombol "Load unpacked".

Window file explorer akan terbuka. Navigate ke folder extension yang sudah Anda extract di Step 2, lalu klik "Select Folder".
